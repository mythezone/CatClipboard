# Icon Files

This directory should contain application icons:

- `icon.png` - Main icon (512x512)
- `32x32.png` - 32x32 icon
- `128x128.png` - 128x128 icon
- `128x128@2x.png` - 128x128@2x icon
- `icon.icns` - macOS icon
- `icon.ico` - Windows icon

You can use tools like:
- https://www.iconfinder.com/ (Free icons)
- https://iconscout.com/ (Free icons)
- https://www.flaticon.com/ (Free icons)

Or generate icons using:
- https://icon.kitchen/
- Photoshop/GIMP/Figma

For now, create a simple clipboard icon with orange/red color scheme (avoiding purple and blue as specified in the design doc).
